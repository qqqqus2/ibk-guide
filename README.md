✔️ 디자인 가이드
모든 간격, 크기, 여백 값은 일관된 디자인 시스템 적용을 위해 가능한 한 4px 단위의 배수 체계(4pt Grid System)를 기준으로 설계하였습니다.
(이에 따라 실제 퍼블리싱 과정에서 일부 간격 값은 디자인 파일과 소폭 차이가 발생할 수 있습니다.)

✔️ 클래스 네이밍 규칙
페이지명__영역명-요소명 형식을 기본으로 사용합니다.

✔️ SCSS 파일 구조
- _var.css : 컬러, 간격 등 공통 변수 관리
- _mixin.scss : 재사용 가능한 믹스인 및 함수 정의
- _reset : 브라우저 기본 스타일 초기화
- _layout.scss : 헤더, LNB, 컨텐츠 영역 등 레이아웃 스타일

✔️ 참고사항
- 본 가이드는 지속적으로 업데이트될 수 있습니다. (260604 현재 디자인 확정되지 않음.)
- 디자인 및 퍼블리싱 규칙은 프로젝트 진행 상황에 따라 일부 변경될 수 있습니다.
